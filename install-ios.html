Installer E-Pointage

Choisissez votre type d'appareil

1

Ouvrez dans Safari

Appuyez sur le bouton bleu ci-dessous. L'app s'ouvrira dans Safari.
⚠️ Ne pas utiliser Chrome ou Firefox pour installer.

Ouvrir dans Safari
2

Appuyez sur "Partager"

En bas de Safari, appuyez sur l'icône (carré avec flèche vers le haut).

3

Sur l'écran d'accueil

Dans le menu qui s'ouvre, faites défiler et tapez sur "Sur l'écran d'accueil" puis confirmez avec "Ajouter".

4

Configurez votre école

Ouvrez l'icône E-Pointage depuis votre écran d'accueil. Entrez votre Code Établissement (ex: ISMEPOINTAGE) et assurez-vous d'être sur le Wi-Fi de l'école.

C'est tout ! L'application fonctionne maintenant comme une vraie app native, même en plein écran, sans la barre d'adresse de Safari.
Conseil : Si l'icône affiche un "E" à la place du logo, supprimez et réajoutez l'application depuis cette page.
1

Téléchargez l'APK

Téléchargez directement le fichier APK depuis le lien ci-dessous. Pas besoin du Play Store.

Télécharger l'APK Android
2

Autorisez les sources inconnues

Si demandé, allez dans Paramètres → Sécurité et activez "Sources inconnues" pour votre navigateur.

3

Installez et configurez

Ouvrez le fichier téléchargé et installez l'application. Au premier lancement, entrez votre Code Établissement.

Application installée ! Vous pouvez maintenant vous connecter avec votre identifiant fourni par l'administration.
