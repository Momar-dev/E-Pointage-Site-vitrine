E-Pointage Mobile

Suivez les étapes ci-dessous pour installer l'application sur votre appareil iOS ou Android.

1

Ouvrez l'Application et entrez votre Code

Cliquez sur le bouton ci-dessous, puis saisissez le code de votre établissement fourni par l'administration.

Lancer l'Application
2

Attendez la redirection

L'application va vous rediriger automatiquement vers le serveur local de votre école. Assurez-vous d'être connecté au Wi-Fi de l'établissement.

3

Ajoutez à l'écran d'accueil

Une fois sur le portail de votre école, appuyez sur Partager puis choisissez "Sur l'écran d'accueil".

Note : Une fois installée, ouvrez l'icône sur votre écran d'accueil et saisissez votre Code École pour vous connecter.
