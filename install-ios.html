E-Pointage Mobile

Suivez les tapes ci-dessous pour installer l'application sur votre appareil iOS ou Android.

1. Ouvrir l'Application
2
Appuyez sur "Partager"

Une fois l'application ouverte, touchez l'icône de partage de votre navigateur (le carré avec une flèche vers le haut).

3
Ajoutez à l'écran d'accueil

Faites défiler les options et choisissez "Sur l'écran d'accueil". Confirmez l'ajout.

Note : Une fois installée, ouvrez l'icône sur votre écran d'accueil et saisissez votre Code École pour vous connecter.
