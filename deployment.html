Guide Complet de Déploiement

Tout ce dont vous avez besoin pour mettre en place E-Pointage dans votre établissement, du matériel aux logiciels.

Infrastructure Établissement

Matériel Requis

Serveur Local : Un PC/Mac (8Go RAM recommandé) pour héberger la base de données.
Réseau Wi-Fi : Un routeur pour assurer la liaison entre les terminaux et le serveur.

Logiciels & Config

Node.js : L'environnement d'exécution pour le moteur de pointage.
Navigateur : Chrome, Safari ou Edge pour l'administration.

Pour les Étudiants

Terminaux

Smartphone : Compatible avec Android (APK) et iOS (iPhone via PWA).

Installation

Android : Téléchargement direct de l'APK depuis ce site.
iOS : Ajout à l'écran d'accueil Safari (sans installation App Store).

Note : Les étudiants reçoivent un Code Établissement unique pour s'enregistrer lors de la première utilisation.

Importation simplifiée

Inutile de saisir les élèves un par un. Préparez simplement un fichier CSV ou Excel avec les informations de vos classes, et importez-les instantanément dans le portail d'administration.

Noms & Prénoms Matricules Filières

Prêt en 5 minutes

Le système génère automatiquement les identifiants pour chaque étudiant dès l'importation.
